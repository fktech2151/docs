## Set up printing on a Chromebook

1. Make sure the client is connected to your organization’s network.
2. Install [the Mobility Print Chrome](https://chrome.google.com/webstore/detail/project-banksia/alhngdkjgnedakdlnamimgfihgkmenbh) app .
3. Open the page to print.
   - The way you print is different depending on the app you’re printing from. For more information, see the documentation for the app. When you select a printer, make sure you select a Mobility Print printer as identified by the PaperCut icon.
4. Print the content.
   - For example, to print from Microsoft Word, press Ctrl+P, select a printer; then click Print.
5. When prompted, sign in.

<Tip>Your Chromebook must run ChromeOS version 69+.</Tip>
